# What's the weather like in Oahu, HI?

## Introduction
Hawaii is an archipelago of [volcanic islands](https://oceanservice.noaa.gov/facts/hawaii.html) in the north Pacific Ocean. It is a popular tourist destination all year round. There are [eight main islands](https://en.wikipedia.org/wiki/List_of_islands_of_Hawaii): Hawai'i, Maui, Kaho'olawe, Lana'i, Moloka'i, O'ahu, Kaua'i, and Ni'ihau. O'ahu is the [third largest](https://traveltips.usatoday.com/8-large-islands-hawaii-108082.html) island in the archipelago and one with varied geographies.
